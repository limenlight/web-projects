Basic style guide for your website

Why is it important to use style on a website?

Adding style to a website, typically accomplished through CSS (Cascading Style Sheets), is essential because it defines the visual appearance and layout of a webpage. This not only makes the site user-friendly and aesthetically pleasing but also ensures consistency with the brand identity. Ultimately, styling enhances the overall user experience by providing a clear structure and a professional look across different pages. The purpose of this webpage is to provide you with a basic understanding of colors, fonts, and text styles in web design.

Colors

There are four ways to represent color in CSS:

  • Named colors:

    • There are more than 140 named colors.

  • Hexadecimal or hex colors:

    • Hexadecimal is a number system that has sixteen digits, 0 to 9 followed by “A” to “F”.
    • Hex values always begin with # and specify values of red, blue, and green using hexadecimal numbers such as #23F41A.
    • Six-digit hex values with duplicate values for each RGB value can be shorted to three digits.

  • RGB

    • RGB colors use the rgb() syntax with one value for red, one value for blue and one value for green.
    • RGB values range from 0 to 255 and look like this: rgb(7, 210, 50).

  • HSL

    • HSL stands for hue (the color itself), saturation (the intensity of the color), and lightness (how light or dark a color is).
    • Hue ranges from 0 to 360 and saturation and lightness are both represented as percentages like this: hsl(200, 20%, 50%).

  • You can add opacity to color in RGB and HSL by adding a fourth value, a, which is represented as a percentage. For example:

Fonts

Fonts play a crucial role in web design for several reasons:

  • Readability: The right font ensures that text is easy to read, enhancing user experience.

  • Aesthetics: Fonts contribute to the overall look and feel of a website, aligning with the brand’s style and message.

  • Hierarchy: Different font sizes and styles help establish a visual hierarchy, guiding users through the content.

  • Emotion and Tone: Fonts can convey emotions and set the tone, whether formal, playful, modern, or traditional.

  • Consistency: Using consistent fonts across a site maintains a cohesive and professional appearance.

Styles

Text style in web design refers to the visual appearance of text, which includes various properties that can be adjusted to enhance readability and aesthetics. Key aspects of text style include:

  • Font Family: The typeface used, such as Arial, Times New Roman, or custom fonts

  • Font Size: The size of the text, typically measured in pixels, ems, or rems.

  • Font Weight: The thickness of the text, ranging from light to bold

  • Font Style: The slant of the text, such as normal or italic.

  • Text Decoration: Additional styling like underlining or line-through.

  • Text Transform: Modifications like uppercase, lowercase, or capitalize.

  • Line Height: The space between lines of text, affecting readability.

These properties help create a visually appealing and readable text layout. For example:
